# El I Chingón - Cuartel General

Este repositorio contiene la infraestructura estática de mi sitio web y el núcleo de datos para el ecosistema de "El I Chingón".

## Propósito del Proyecto
Un espacio digital soberano dedicado a la investigación e interpretación del I Ching, su relación con la ciencia de sistemas, la filosofía y la tecnología moderna.

## Arquitectura Técnica
* **Alojamiento:** GitHub Pages.
* **Estructura:** Modular por silos indexables.
* **Idiomas:** Gestión multilingüe mediante sufijos ISO de archivos.
